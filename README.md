# Lurking In The Shadows - Web App

Aplicación web complementaria para el videojuego de terror **Lurking In The Shadows**, un juego de Backrooms desarrollado en Unity.

## Stack

- **Frontend:** Angular 21 (SCSS)
- **Backend:** Spring Boot 3.5 (Java 21)
- **Base de datos:** Supabase (PostgreSQL)

## Funcionalidades

- Registro e inicio de sesión de usuarios
- Sistema de reseñas con puntuación
- Comentarios con respuestas
- Página de descarga del juego (sin login)

## Requisitos

- JDK 21
- Node.js 18+
- Angular CLI

## Desarrollo

```bash
# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
ng serve
```
